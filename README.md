# Friday AI Voice Chatbot

Friday is a full-stack AI chatbot web application that allows users to interact with an AI assistant through a modern chat interface. The chatbot can respond to user messages in real time and also speak responses using a realistic female voice.

## Features

* Real-time chat interface
* AI generated responses
* Voice response using ElevenLabs API
* Typing indicator for bot replies
* Clean and interactive UI
* Full stack architecture (frontend + backend)

## Tech Stack

Frontend:

* HTML
* CSS
* JavaScript

Backend:

* Python
* Flask

AI & Voice:

* ElevenLabs Text to Speech API

## How to Run the Project

1. Clone the repository

git clone https://github.com/bhavishya-ai-crypto/friday-ai-chatbot.git

2. Go to the project folder

cd friday-ai-chatbot

3. Install backend dependencies

pip install flask flask-cors

4. Run the backend server

python app.py

5. Open the frontend

Open `frontend/index.html` in your browser.
